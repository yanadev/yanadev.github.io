PHP 与 HTML 混编

PHP与HTML混编​

普通嵌入

<p><?php echo "hello"; ?></p>

语句混编

<?php if($age>=18){ ?>
 <p>成年人</p>
<?php }else{ ?>
 <p>小孩子</p>
<?php } ?>

指令式语句 : 将 { 、 } 换成 :

 <?php if($age>=18): ?>
  <p>成年人</p>
 <?php else: ?>
  <p>小朋友</p>
 <?php endif ?>
