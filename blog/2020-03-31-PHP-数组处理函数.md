## 获取关联数组中全部的键 / 值

```php
array_keys() / array_values()
```

## 判断关联数组中是否存在某个键

```php
array_key_exists()
```

## 去除重复的元素

```php
array_unique()
```

## 将一个或多个元素追加到数组中

```php
array_push() $arr[] = 'new value'
```

## 删除数组中后一个元素

```php
array_pop()
```

## 数组长度

```php
count()
```

## 检测存在

```php
in_array()
```
