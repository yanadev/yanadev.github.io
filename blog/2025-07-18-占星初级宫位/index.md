# 穿刺

当星体落入某一宫，距离下一个宫头 **5 度内**，自动穿刺进入下一宫，只能顺穿不能逆穿

穿刺星体兼具落入两个宫位的特征，比例一半一半

# 三王星入宫位

落宫原则

1. 三王没有分数，所以必须考虑相位
2. 三王和别的星体呈现吉相位，在落入的宫位体现自身吉星
3. 三王和别的星体呈现凶相位，在落入的宫位体现自身凶性
4. 三王和别的星体合相时候，跟随合相星的状态
5. 体现原则：首先体现三王所在宫位，其次作为工具影响与其产生相位的星体的宫位
