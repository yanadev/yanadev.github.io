# 宏观概念

1. 一象百应： 一个特质可能在不同人上应征不同的象

2. 切蛋糕：一个星体对应多种象，但是一旦多个符合的出现，后续出现的可能性降低，太阳能量已经被分走了

3. 情商看月亮、金星、水星，月亮看反应，金星看理解，水星看表达

4. 太阳与木星的区别，太阳象征着某个团体内部的位阶高，家庭内部长辈，公司内部领导；木星象征社会层面上的位阶高，老师、牧师
   等，不一定有直接关联

# 相位原则

1. 优先看紧密相位即精准相位，越紧密优先度越高，作用力越大
2. 重点观测和命主星产生的相位，如果命主星空相，则考虑定位星
3. 抓主要相位，保持自身论断系统的简洁
4. 同一颗星体，同时拥有吉相位、凶相位时候，不可以互相抵消，都会体现，要做到趋吉避凶，充分利用吉相位的利好有效消减凶相位
   带来的影响

- 灼伤 行星距离太阳 17′-8°30′，并在同一星座，受到太阳的压制与遮蔽，无法顺畅展示自身能力
- 在日光下，行星距离太阳 8°30′-17° 之间，距离稍远，不像灼烧那么严重，但是也不理想

## 星体能量博弈理论，看星体的分数

- 合相：正正吉相叠加，负负凶相叠加，正负不中和，各应各的
- 相冲、相刑克：正正相互损耗，负负泥潭里越陷越深，正负博弈谁大谁赢，相刑克比相冲力量弱一点
- 三合、六合：正正相互帮助，负负惺惺相惜，正负则是正帮助负，六合比三合能量弱一点
- 三王不参与，只做工具使用，作用于其他星体，被作用的星体，正分吉相为吉，正分凶相为减吉，负分吉相为减凶，负分凶相为加强凶
  相，合相视情况而定

1. 日月合相 - 0° 新月

- “人生动力” 与 “感情动力” 的结合
- 太阳代表我所追求的，月亮表示我所需要的
- 我追求的正是我所需要的

- 新月出生的人决断、自强、不妥协、目标明确，但是如果发展过于负面，会出现自负、自大的特征
- 日月合代表父母角色的统一性，生活态度、价值观、人生目标、教育方式等非常相识
- 如果恰巧出现在两个星座，就不会统一，貌合神离

2. 日月相冲 - 180° 满月

- “意志” 与 “感情” 的冲突
- 有徘徊不定的倾向，婚姻生活可能会有紧张和冲突（会体现在父母、自己的婚姻，需要配合切蛋糕、宫位、婚姻宫和父母宫）

- 满月的人性格不够坚定，犹豫不决，对别人的态度、感觉、需求等高度敏感
- 正面特质：知道如何将一对一的关系服务于自己，避免新月的一根筋，一件事有多种可能性
- 负面特质：过于在乎激烈的人际关系，而忘记了本身要去做的事情，父母截然不同，这种反差导致孩子在选择哪一方作为榜样感觉困惑
  ，即便成年了也在心灵深处整合这两种对立的原型

3. 日月刑克 - 90° 上弦月与下弦月

- “自我” 与 “情感” 的矛盾
- 在朝向自己的目标努力时受到阻碍
- 月亮代表的情绪问题和习惯模式阻碍了自我表达；月亮代表家庭，家庭的影响也阻碍了自我的发展
- 日月阴阳能量不融洽，导致他们对异性缺乏了解和相处技巧等问题
- 父母方面，刑克比对冲更为拧巴，因此对比满月，可能出生在一个充满改变和冲突的家庭，父母相互拧巴，孩子感觉自卑、自责，父母
  关系的紧张状态迫使他们去尝试更好的处理方法，生存在自己的家庭之中

4. 日月三合 - 120° 和谐相位

- 意志与习惯、情感相协调，过去和未来不存在冲突
- 童年受到父母的良好教育和样板引导，具有健康的自我，人际关系轻松自如，能够承担责任和得到认可，能够让别人听从自己而无需强
  迫，自足，心态很好，但难以对自己的人生进行疑问和思辨

5. 日月六合 - 60° 和谐相位

- 和谐相位，作用比三分稍弱，一样代表平静安逸的生活，婚姻、家庭和谐自如
- 心态平衡，积极，能汲取过去的教训又不拘泥于过去的记忆，这样的人坚定，又会合适的妥协
