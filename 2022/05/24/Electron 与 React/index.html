Chromium

Chromium 是 Google 的开源浏览器（Chrome 的不太稳定更新快的兄弟版）

组合

该框架创建的应用使用 网页 作为其 GUI，可看做是 由 JS 控制的迷你精简版 Chromium 浏览器

也可看做是 将 Electron 当做 node.js 的变体，只是更加专注于桌面应用而非 web 服务器

  • package.json 设定的 main 脚本的所在进程称作是 主进程
    • 该进程中运行的就脚本可通过创建网页的方式来展现其他 GUI
  • 由于 Electron 通过 Chromium 来显示页面，所以 Chromium 自带的多进程架构也一同被利用
    • 这样每个页面都运行着一个独立的进程，统称为 渲染进程
  • 通常来说，浏览器中的网页会被限制在沙盒环境中运行，且不允许访问系统原生资源
    • 但是由于 Electron 用户可在页面中调用 node.js API，所以可以和底层操作系统直接交互

优劣

优点

  • 方便快捷的开发桌面应用
  • 跨平台
  • 对前端开发者友好
  • 活跃的社区
  • 丰富的 API
  • …

缺点

  • 性能比不上原生的桌面应用
  • 发布的包有一点臃肿（包裹了整个 Chromium）
