Electron - Menu 模块

发表于 2022-05-25 分类于 Electron ， Menu 模块 阅读次数：
本文字数： 703

Menu 主进程模块

  • 创建原生菜单：可作为 应用菜单 和 context 菜单

  • 每个菜单下面都有 一个或几个菜单项 menu items，且每个菜单项都可以有自己的子菜单

通过 remote 模块动态创建菜单

类 Menu（静态方法）

创建菜单 new Menu()

Menu.setApplicationMenu(menu) 设置菜单

  • 在 OS X 上设置应用菜单 menu
  • 在 windows、 linux ，是为每一个窗口都在其顶部设置菜单 menu

Menu.setActionToFirstResponder(action) OS X 发送 action 给应用的第一个响应器，用来模仿 Cocoa 菜单的默认行为，通常只需要用到 MenuItem 的属性 role

Menu.buildFromTemplate(template) 创建菜单

实例方法

menu.popup([browserWindow, x, y, positioningItem]) 弹出菜单并指定菜单位置和索引

  • browserWindow 可选 默认 null
  • x Number 可选 默认-1
  • y Number （若x设置了，必填） 默认-1
  • positioningItem Number 可选 在指定鼠标位置下面的菜单项的索引，默认-1

在 BrowserWindow 中弹出 context menu ，可选择提供指定的 x,y 来设置菜单位置，否则默认放在当前鼠标位置

menu.append(menuItem) 添加菜单项

menu.insert(pos, menuItem) 在指定位置添加菜单项

menu.items() 获取菜单项数组

菜单项位置

  • before 菜单项 id 之前插入，若不存在，插入末尾
  • after i菜单项 id 之后插入，若不存在，插入末尾
  • endof 集合末尾插入
