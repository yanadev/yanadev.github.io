hexo博客的使用

发表于 2019-10-16 更新于 2022-05-23 分类于 hexo 阅读次数：
本文字数： 459

命令：

  • hexo clean :清除数据

  • hexo g：生成博客

  • hexo s：启动博客

  • hexo d：部署到远端

踩坑

  • 遇到node非内部外部命令，有可能是nodejs文件改变位置，相应的环境变量path没有改变
  • 遇到除了主页之外其他页面头像显示不正常，是因为yilia的配置文件，关于avatar的路劲配置错误，要在img之前加反斜杠

将blog部署到GitHub上

在个人的GitHub网站上，新建一个repository（仓库），输入符合的要求的名字

之后再在之前创建的myBlog目录下面，安装git部署插件

  • mac：cnpm install –save hexo-deployer-git
  • win：npm install hexo-deployer-git –save

安装完成之后，修改myBlog目录下的_config.yml文件

每次上传博客流程，hexo clean=====hexo g====hexo s(test)=====hexo d(transport)
