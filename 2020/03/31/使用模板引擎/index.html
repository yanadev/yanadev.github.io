使用模板引擎

发表于 2020-03-31 更新于 2022-05-23 分类于 模板引擎 阅读次数：
本文字数： 3.1k

使用步骤

  1. 选择一个模板引擎

  2. 下载模板引擎 JS 文件 https://unpkg.com/art-template@4.13.2/lib/template-web.js

  3. 引入到页面中

  4. 准备一个模板

  5. 准备一个数据

  6. 通过模板引擎的 JS 提供的一个函数将 模板 和 数据 整合，得到渲染结果 HTML

数据: data.php

重点

script 标签的特点

  1. innerHTML 永远不会显示在界面上
  2. 如果 type 不等于 text/javascript 的话，内部的内容不会作为 JavaScript 被解析

不在 JS 变量中写模板的原因

如果将模板写到 JS 中，维护不方便，不能换行，没有着色

使用 script 标签的写模板的原因

script 不会将内容渲染到页面上
