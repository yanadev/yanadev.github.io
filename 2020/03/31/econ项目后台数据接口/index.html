econ项目后台API接口文档

API 接口说明

  • 接口基准地址： http:127.0.0.1:8080/admin/api
  • 服务端已开启 CPOS 跨域支持
  • 使用 HTTP Status Code 标识状态
  • 数据返回格式统一使用 JSON

支持的请求方法

  • GET：从服务器取出资源（一项或者多项）
  • POST：在服务器新建、删除、获取、修改一个资源

通用返回状态说明

状态码 含义 说明
200 OK 操作成功
400 BAD REQUEST 请求的地址不存在或者包含不支持的参数
401 UNAUTHORIZED 未授权
403 FORBIDDEN 被禁止访问
404 NOT FOUND 请求的资源不存在
422 Unprocesable entity 当创建一个对象时，发生一个验证错误

登录

登录验证接口

  • 请求路径：admin/api/login.php
  • 请求方法：post
  • 请求参数
参数名 参数说明 备注
email 邮箱 不能为空
password 密码 不能为空
  • 相应参数
参数名 参数说明 备注
status 状态码
msg 状态信息
token 令牌 使用md5加密用户信息存储在数据库
  • 响应数据

用户管理

用户数据列表

  • 请求路径：admin/api/users.php
  • 请求方法：get
  • 请求参数
参数名 参数说明 备注
query 查询参数 可以为空
pagenum 当前页码 不能为空
pagesize 每页显示条数 不能为空
  • 响应参数
参数名 参数说明 备注
pagenum 当前页码
total 总数据条数
msg 响应信息
status 响应码
users 用户数据 索引数组
  • 响应数据

修改用户状态

  • 请求路径：/admin/api/users-status.php
  • 请求方法：post
  • 请求参数
参数名 参数说明 备注
id 用户 ID 不能为空，参数形式params
status 用户状态 不能为空携带在params中，值为 activated 或者 unactivated
  • 响应数据
