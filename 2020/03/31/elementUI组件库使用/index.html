基于图形化界面自动安装

① 运行 vue ui 命令，打开图形化界面

② 通过 Vue 项目管理器，进入具体的项目配置面板

③ 点击 插件 -> 添加插件，进入插件查询面板

④ 搜索 vue-cli-plugin-element 并安装

⑤ 配置插件，实现按需导入，从而减少打包后项目的体积
