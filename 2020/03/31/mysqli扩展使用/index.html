mysqli扩展使用

发表于 2020-03-31 更新于 2022-05-23 分类于 MySQL ， 数据库 阅读次数：
本文字数： 911

连接数据库

成功 返回 数据库连接对象，失败返回 false 及warning

执行 SQL 语句

结果集模式有两种常量

MYSQLI_STORE_RESULT：全部读取到 PHP 端

MYSQL_USE_RESULT：仅初始化结果集检索，处理结果集的时候进行数据读取

处理结果集：从结果集中获取信息

function description
mysqli_num_rows() 获取行数
mysqli_fetch_all() 获取所有结果，以数组形式返回
mysqli_fetch_array() 获取一行结果，以数组形式返回
mysqli_fetch_assoc() 获取一行结果，以关联数组形式返回
mysqli_fetch_row() 获取一行结果，以索引数组形式返回

其他操作函数

function description
mysqli_insert_id() 获取上一次插入操作时产生的ID
mysqli_affected_rows() 获取上一次操作时影响的行数
mysqli_real_escape_string() 转义SQL语句字符串中的特殊字符
mysqli_error() 返回最近函数调用的错误代码
mysqli_free_result() 释放结果集
mysqli_close() 关闭数据库连接

  > mysqli_free_result()函数用于释放结果集占用的系统内存资源
  >   >
  >   >   > mysqli_close()函数用于释放打开的数据库连接
