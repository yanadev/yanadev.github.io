  1. 行内元素

  • width、height无效，可通过line-height设置

  • margin、padding左右有效，上下无效

2. float
  • none：设置对象不浮动
  • left：设置对象浮在左边
  • right：设置对象浮在右边
3. clear
  • none：允许两边都可以有浮动对象
  • both：不允许有浮动对象
  • left：不允许左边有浮动对象
  • right：不允许右边有浮动对象
4. position （子绝父相）
  • static：常规留
  • relative：参照自身
  • absolute：参照父元素，无父元素则参照body
  • fixed：以窗口为参照
5. lie-height特殊使用
  • 行高会继承，因此直接给父级，当line-height等于height的时候垂直居中
